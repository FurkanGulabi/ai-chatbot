Project: AI Chatbot

Overview:
This is an AI Chatbot project built with Next.js, React (server components), and integrated with Prisma and next-auth for authentication. The project leverages advanced UI components built with shadcn/ui, Tailwind CSS for styling, Sonner for toast notifications, and next-themes for dark mode support. It uses separate layouts for Chat and Authentication, featuring a responsive sidebar with off-canvas behavior for mobile. The overall architecture is designed to provide a modern, interactive chat experience with scalable backend integration.
